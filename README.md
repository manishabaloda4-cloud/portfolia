📌 What is BizAgent AI?

BizAgent AI is a multi-agent business assistant built specifically for Indian SMBs (small and medium businesses). It helps business owners automate:


📞 Customer inquiry handling — auto-respond to common questions
📦 Order management — track and update orders via AI
📊 Business analytics — simple insights from sales data
🗓 Appointment booking — automated scheduling for service businesses



🏗 Agent Architecture

BizAgent Orchestrator
    │
    ├──► Customer Support Agent
    │         └─ Handles FAQs, product queries, complaints
    │
    ├──► Sales Agent  
    │         └─ Lead qualification, follow-ups, quotes
    │
    ├──► Analytics Agent
    │         └─ Revenue reports, top products, trends
    │
    └──► Booking Agent
              └─ Appointment scheduling, reminders
